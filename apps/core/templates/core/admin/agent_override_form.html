Help
What are Overrides?

Overrides allow you to set custom commission rates for specific agents, overriding the default commission structure.

When to Use Overrides
  • Rewarding top-performing agents
  • Temporary promotional rates
  • Special agreements with specific agents
  • Correcting commission discrepancies
Priority

Agent overrides always take precedence over commission structures. If an override exists for an agent and policy type, it will be used.
